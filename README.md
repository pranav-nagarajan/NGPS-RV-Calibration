# Measuring Radial Velocities with the Next Generation Palomar Spectrograph

This repository serves as a tutorial for computing flexure corrections for and radial velocities (RVs) with the Next Generation Palomar Spectrograph (NGPS) at Palomar Observatory. This method of RV determination is calibrated using RV standard stars, and will be described in detail in Nagarajan et al. (forthcoming).
